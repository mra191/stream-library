# Streaming Playlist

## 📺 Apa itu TIPI.m3u?

**TIPI.m3u** adalah kumpulan link streaming untuk menonton **siaran langsung TV Indonesia** dan channel internasional berkualitas tinggi. File ini berisi playlist M3U yang kompatibel dengan berbagai aplikasi streaming.

---

## 📻 Apa itu radio.m3u?

**radio.m3u** adalah kumpulan link streaming untuk mendengarkan **stasiun radio Indonesia** dari berbagai genre dan kategori. File ini berisi playlist M3U dengan channel radio musik, berita, dan hiburan berkualitas.

---

## 🚀 Cara Menggunakan

### 1️⃣ **Dengan VLC Media Player**
```
File → Open Network Stream → Paste URL dari TIPI.m3u atau radio.m3u
```

### 2️⃣ **Dengan Aplikasi IPTV**
- Smart TV IPTV Apps
- Kodi
- TiviMate
- Perfect Player
- IPTV Smarters Pro

### 3️⃣ **Untuk Radio**
- Aplikasi radio streaming
- VLC Media Player
- Android radio apps
- Web players yang support M3U

### 4️⃣ **Direct Link**
Copy URL channel favorit Anda dan paste ke aplikasi pemutar yang mendukung format HLS/M3U8

---

## 📋 Format File

File ini menggunakan format **M3U** standar:
```
#EXTM3U
#EXTINF:-1, [Nama Channel]
[URL Stream 1]
[URL Stream 2 (Backup)]
```

## ⚠️ Catatan Penting

🔗 **Link dapat berubah** - Beberapa streaming link mungkin tidak selalu aktif atau stabil  
📡 **Kecepatan Internet** - Minimal 5 Mbps untuk streaming HD  
🎬 **Backup Links** - Beberapa channel memiliki link alternatif jika yang utama down  
⏱️ **Update Berkala** - Playlist diperbarui sesuai ketersediaan link streaming

---

## 🛠️ Troubleshooting

**Link tidak bisa diputar?**
- Coba link alternatif (jika tersedia)
- Periksa koneksi internet Anda
- Gunakan VPN jika diperlukan

**Buffer atau buffering?**
- Kurangi kualitas streaming
- Tutup aplikasi lain yang menggunakan bandwidth
- Coba link backup yang tersedia
